# newspaper-agency2

Python3 must be already installed

```shell
git clone https://github.com/asdadaversa/newspaper-agency2.git
cd newspaper-agency
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
```
